<h1>Installation</h1>

<pre>
	<code>
1-composer install or update
2-npm install
3-config database (env.exemple to .env)
4-php artisan key:generate
5-php migrate --seed
6-php artisan serve (http://127.0.0.1:8000)
	</code>
</pre>
